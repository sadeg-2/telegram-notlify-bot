# Telegram Notlify Bot

This is a Telegram bot that sends random motivational messages using the OpenAI API with cron job scheduling.

## Features
- Sends motivational messages to users.
- Utilizes the OpenAI API for generating content.
- Scheduled messages with cron jobs.
